Data Security Classification Pyramid

A four-level pyramid showing data security classification.
- L4 (Top, Red): "绝密 (Top Secret)" - Icon: Lock or Shield. Examples: Core Algo, State Secrets.
- L3 (Orange): "机密 (Secret)" - Icon: Confidential Folder. Examples: Customer List, Financials.
- L2 (Yellow): "内部公开 (Internal)" - Icon: Office Building. Examples: Employee Handbook.
- L1 (Base, Green): "公开 (Public)" - Icon: Globe/Public. Examples: Website Info.
