# 工具选型矩阵 (Selection Matrix)

## 视觉描述
一个四轴坐标图。
- **纵轴**: 预算 (从低到高)。
- **横轴**: 业务复杂度 (从低到高)。
- **区域划分**: 
    - 低预算/低复杂: 开源组合。
    - 高预算/高复杂: 华为重装。
    - 快速敏捷需求: 阿里云。
- **细节**: 每个象限配一个复古的指南针指明方向。

## 标签文字 (Labels)
- 选型指南 (Selection Guide)
- 预算匹配 (Budget)
- 复杂度 (Complexity)
- 最优解 (Optimal)
