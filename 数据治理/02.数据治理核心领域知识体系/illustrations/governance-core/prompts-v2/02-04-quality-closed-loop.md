# 质量闭环管理 (Closed-loop Quality Management)

## 视觉描述
展现一个无限循环的治理过程。
- **风格**: 机械原理图风格。
- **环节**: 
    1. **发现**: 一个放大镜对准一排数据列。
    2. **归因**: 机械臂拆解问题部件，标注“根因分析”。
    3. **整改**: 工人修理图标，清理污垢。
    4. **复盘**: 一张记分卡和打勾标志。
- **循环**: 箭头连接成环，中心标注“持续提升”。
