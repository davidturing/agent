# 元数据与血缘 (Metadata & Lineage)

## 视觉描述
一棵数据之树，根部深入泥土，枝叶连接云端。
- **根部**: 标注为“技术元数据 (Technical)”，如表名、字段、类型。
- **主干**: 标注为“业务元数据 (Business)”，如业务术语、定义。
- **枝叶**: 标注为“操作元数据 (Operational)”，如任务运行时长、报错日志。
- **血缘**: 在树叶间有一条红色的经脉式连线，展示从原始数据到最终报表的端到端流转。
