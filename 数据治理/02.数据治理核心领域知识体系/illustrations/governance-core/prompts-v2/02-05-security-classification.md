# 数据安全分级分类 (Security Classification)

## 视觉描述
一个类似文件柜或阶梯的金字塔结构，展示不同密级的数据。
- **风格**: 复古打字机/办公室档案室风格。
- **层级**:
    - 底部: **公开 (Public)** - 报纸图标。
    - 中下: **内部 (Internal)** - 办公文档图标。
    - 中上: **秘密 (Confidential)** - 带锁的文件夹图标。
    - 顶部: **绝密 (Top Secret)** - 带有指纹扫描和严密封条的保险箱。
- **标注**: 右侧显示对应的业务分类示例（如：人力资源、财务、技术）。

## 标签文字 (Labels)
- 绝密 (Top Secret)
- 秘密 (Confidential)
- 内部 (Internal)
- 公开 (Public)
- 分级分类 (Classification)
- 权限控制 (Access Control)
