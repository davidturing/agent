# ETL 与 ELT 对比 (ETL vs ELT)

## 视觉描述
两条工业流水线。
- **上行 (ETL)**: 采集 -> 经过一个复杂的“处理工厂 (Transform)” -> 存入仓库。
- **下行 (ELT)**: 采集 -> 直接存入大型“数据湖 (Load)” -> 在湖内利用算力进行“处理 (Transform)”。
- **风格**: 二十世纪初期的工厂线稿图，结合现代服务器图标。

## 标签文字 (Labels)
- 传统方式 (Traditional)
- 云原生方式 (Cloud Native)
- 采集 (Extract)
- 转换 (Transform)
- 加载 (Load)
- 算力下推 (Push-down)
