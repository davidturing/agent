# 架构对齐 (Architecture Alignment)

## 视觉描述
一张展示业务视角、数据视角与技术视角对齐的示意图。采用复古蓝图风格。
- **背景**: 略微泛黄的工程图纸背景。
- **主体**: 三层平行重叠的半透明网格面板，从上到下依次为：
    1. **业务层**: 包含“销售”、“研发”、“采购”等齿轮图标。
    2. **数据层**: 展示其间的连线，节点标注为“客户模型”、“物料标准”。
    3. **技术层**: 底部支撑，显示“ERP”、“PLM”、“数据湖”的服务器图标。
- **标注**: 使用手写风格的中文标签。
