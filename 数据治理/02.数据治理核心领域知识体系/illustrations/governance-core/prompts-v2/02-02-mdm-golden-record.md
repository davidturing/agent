# MDM 黄金记录 (MDM Golden Record)

## 视觉描述
展示多个遗留系统中的冲突数据汇聚成一个唯一的、正确的“黄金记录”的过程。
- **风格**: 50年代工业设计感。
- **左侧**: 三个破旧的监视器屏幕，分别代表 ERP、CRM、OA。每个屏幕上显示的“张三”信息都有细微差别（名字拼写、电话不同）。
- **中间**: 一个漏斗状的机械装置，标注为“清洗与合并”。
- **右侧**: 一个闪闪发光的记录卡片，被称为“黄金记录 (Golden Record)”，信息完整且唯一。
- **标注**: 均为中文。

## 标签文字 (Labels)
- 源系统 (Source Systems)
- 冗余与冲突 (Redundancy & Conflict)
- 数据合并 (Data Merging)
- 唯一真相 (Single Source of Truth)
- 黄金记录 (Golden Record)
- 客户 ID (Customer ID)
