# 静态与动态脱敏 (SDM vs DDM)

## 视觉描述
对比两种数据脱敏方式。
- **背景**: 实验室实验台。
- **左侧 (静态 SDM)**: 一个大水缸，里面的水被永久染黑。标注为“存储即脱敏”。
- **右侧 (动态 DDM)**: 一个透明水箱，水在流出水管的瞬间变黑。标注为“查询时脱敏”。
- **细节**: 水管末端是一个带眼镜的分析师，他看到的都是脱敏后的数据。
