# MicroBlocks Articles

This archive contains articles scraped from [MicroBlocks Learn](https://learn.microblocks.fun/en/index).

## How to use

1.  **Import to WordPress**: You can use a WordPress importer plugin that supports Markdown or convert these files to HTML/XML for import.
2.  **Manual Publishing**: Open each file, copy the content, and paste it into your WordPress editor.

## Content

Each file corresponds to an activity/article.
Images are linked remotely to the original source.
Videos are linked as requested.

## Metadata

Each file includes YAML frontmatter with:
- title
- original_url
- slug
