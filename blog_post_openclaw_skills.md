# Latest OpenClaw Skill List

**Date:** February 5, 2026  
**Category:** AI, Automation, Open Source

The OpenClaw ecosystem continues to grow at an incredible pace. With over **1,715 community-built skills** now available, OpenClaw (formerly known as Moltbot) has solidified its position as a versatile and powerful local AI assistant.

Whether you are a developer looking to automate your workflow, a researcher needing deep web integration, or just someone who wants to control their smart home with natural language, there is likely an OpenClaw skill for you.

We've analyzed the [Awesome OpenClaw Skills repository](https://github.com/VoltAgent/awesome-openclaw-skills) to bring you a curated overview of what's available.

## Key Categories

The skills are organized into diverse categories, reflecting the wide range of applications for local AI agents:

### 1. Web & Frontend Development
For web developers, OpenClaw offers tools like **Next.js** helpers, **UI/UX auditing** skills, and integrations with **React Email** and **Tailwind**. You can even find skills to generate entire frontend components or audit your site for accessibility.

### 2. Coding Agents & IDEs
This is a massive category containing tools to supercharge your coding.
- **Computer Use**: Grant your agent full desktop control on Linux.
- **Coding Agents**: Integrate with Cursor, VS Code, and various LLM-based coding assistants.
- **Documentation**: Tools to automatically read and generate documentation.

### 3. DevOps & Cloud
Manage your infrastructure directly from your chat interface.
- **Cloud Providers**: extensive support for **AWS**, **Azure**, **Digital Ocean**, and **Google Cloud**.
- **Containerization**: Manage **Docker** containers and **Kubernetes** clusters (including autoscaling and backups).
- **Monitoring**: Tools for tracking server health and logs.

### 4. Moltbook: The Social Network for Agents
One of the most fascinating developments is **Moltbook**, a social network built specifically for AI agents. Skills in this category allow agents to:
- Post updates and interact with other agents.
- "Hire" other agents for tasks.
- Share knowledge in a decentralized way.

### 5. Productivity & Daily Life
OpenClaw isn't just for coding.
- **Smart Home**: Control IoT devices and Home Assistant.
- **Finance**: Track expenses and crypto markets.
- **Media**: Control music players (even Winamp!) and manage downloads.

## How to Get Started

All these skills are sourced from [ClawHub](https://www.clawhub.com/), OpenClaw's public skills registry. You can browse the full list and instructions on how to install them at the [Awesome OpenClaw Skills GitHub repository](https://github.com/VoltAgent/awesome-openclaw-skills).

Installation is usually as simple as:
```bash
npx clawhub@latest install <skill-slug>
```
